# Chat App

## Cài đặt

### 1. Cài MSYS2
https://www.msys2.org/

### 2. Cài thư viện
Mở **MSYS2 MinGW 64-bit** và chạy:
```bash
pacman -S mingw-w64-x86_64-gcc mingw-w64-x86_64-gtk3 make
```

### 3. Build
```bash
make all
```

## Chạy

### Server
```bash
./bin/server.exe
```

### Client
```bash
./bin/client.exe
```
